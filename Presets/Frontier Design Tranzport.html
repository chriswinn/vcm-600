Overview

This document describes how you can use the TranzPort Remote with applications that aren't supported "out of the box". This is achieved by using Bome's Midi Translator: it converts the MIDI data from the TranzPort to keystrokes which are used to control the various applications. The mapping rules are organized in different presets, one preset per application to be controlled.

General Setup

For any of these presets to work, verify the following steps:
  • Connect the TranzPort Bridge to the computer
  • Start Bome's Midi Translator (MT)
  • In MT's menu, select TranzPort in MIDI IN, and also select it as MIDI OUT
  • In MT's Options menu, unselect MIDI Thru
  • In MT, open the file "Frontier Design TranzPort" (if not already done)
Now the TranzPort's display should display "Midi Translator".

Use the Solo button to cycle through the different presets (Shift+Solo will cycle backwards).
That's it for the setup!

Nullsoft Winamp

Using this preset, you can control Nullsoft's media player Winamp conveniently from the wireless TranzPort. This will work even if Winamp is not active!

After following the setup, start Winamp and bring up its preferences (e.g. with Ctrl+P). There, select the page "Global Hotkeys" and enable them by checking the Enabled checkbox.

Then select the Winamp preset in Bome's Midi Translator by pressing the Solo button until the TranzPort's display shows "Winamp".

Now you can use the following controls on the TranzPort:
TranzPort
control		Function in Winamp
Play buttonStart Playback
Stop buttonStop Playback
Rewind << buttonWind backward (hold to continue winding)
FFwd >> buttonWind forward (hold to continue winding)
Track < buttonPrevious Song
Track > buttonNext Song
Mute buttonToggle: mute/unmute volume (mute LED will light up)
WheelScrub forward and backward
Shift+WheelVolume (will be ignored in muted state)


Windows Media Player

Using this preset, you can operate Windows Media Player (WMP) from the TranzPort Remote.

First, follow the general setup. Then, for convenience, you can define a keyboard shortcut to start WMP: Locate WMP in the start menu (it's in Programs|Accessories|Entertainment), then right-click on the entry in the start menu and select Properties. In the Shortcut Key field, press these keys at the same time: Alt+Ctrl+Shift+P. Click on OK. You can test it - that key combination should now start WMP.

Now switch to Bome's Midi Translator and press the Solo button until the TranzPort's display shows "Windows Media Player".

Now you can use the following controls on the TranzPort:
TranzPort
control		Function in WMP
Play buttonStart Playback
Stop buttonStop Playback
Rewind << buttonWind backward (this doesn't always work in WMP)
FFwd >> buttonWind forward (this doesn't always work in WMP)
Track < buttonPrevious Song
Track > buttonNext Song
Mute buttonToggle: mute/unmute volume
WheelVolume (will unmute if in muted state)
Shift+WheelScrub forward and backward
Loop buttonToggle repeat on/off
Add buttonAdd Tracks window
In buttonNormal/full size mode
Out buttonSkin mode
Punch buttonToggle full screen

Controlling MP3 Direct Cut

Using this preset, you can use the TranzPort to operate MP3 Direct Cut, a useful MP3 editor that lets you edit mp3 files without re-encoding the file, so it keeps the original quality.

After following the setup, select the MP3DirectCut preset in Bome's Midi Translator by repeatedly pressing the Solo button until the TranzPort should displays "MP3DirectCut".

Now you can use the following controls on the TranzPort:
TranzPort
control		Function in MP3DirectCut

Stop buttonStop Playback
Rewind << buttonWind backward
FFwd >> buttonWind forward
Undo buttonUndo last action
In buttonSelect Begin of selection
Out buttonSelect End of selection
Punch buttonDelete selection
Loop buttonListen AS CUT: listen to 4 seconds before beginning of selection, and to 4 seconds after selection.
Shift+In buttonZoom In
Shift+Out buttonZoom Out
Prev buttonGo to previous cut/edit point
Next buttonGo to next cut/edit point
WheelScrub forward and backward one frame at a time
Shift+WheelScrub forward and backward in larger steps


Create Your Own Presets

If the program you want to control is not on this list, or if you want to modify these provided presets, no problem! With Bome's Midi Translator Pro or Author you can create your own presets. It's always a good idea to inspect these existing presets to see how it's done. The documentation that ships with the TranzPort unit includes the PDF file "TranzPort native" which has a useful reference of the MIDI messages sent by the TranzPort device.

The project file has one preset Convert to Absolute Controller which demonstrates how you can use the rules to convert the wheel to output a continuous controller as MIDI data.

With the Author edition of Bome's Midi Translator you can even create signed and/or encrypted presets that you can publish or sell for usage with the free MT Player.
