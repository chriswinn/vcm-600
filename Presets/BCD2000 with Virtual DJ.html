BDC2000 Dj Solution for VDJ v1.0beta

Short guide (i will post full guide later) (only english now - italian later)

Presettings: Go to BCD2000 Control Panel on PC and set Advanced Mode.

If you are not using last driver from Behringer donwload it here http://www.behringerdownload.de/B-Control_download/BCD2000-driver-setup-1-1-1-0.exe

Software needed:

BCD2000 Dj Solution for VDJ v1.0beta - Download it here http://www.macfruits.com/bcd2000/BCD2000n.zip 

  It contains a .bmtp file for Midi Translator
  A .xml file for VirtualDJ
  A BMP with assigned control of remapped bcd2000 [1]

MidiYoke (freewere) - Go to web pages http://www.midiox.com/myoke.htm

Midi Translator (shareware) - Go to web pages http://www.bome.com/products/miditranslator/ 

  Now UPDATED: configuration file in 1.40 format only.
  I advice to use new version of midi translator, many improvements are made.

Install MidiYoke and restart PC.

Install Midi Translator, set BCD2000 as midi in and set MidiYoke 1 as midi out.

Enalble Midi Thru in Midi Translator.

Import configuration file (.bmtp - only for v1.40) and save.

Open VirtualDJ go to setting and enable midi in External Hardware sheet.

Go to Shortcuts sheet (in settings windows) and click to load and select configuration file .xml

Click ok! Enjoy.

If there's some problems try to restar your pc. Antonello Migliorelli.

Audio device settings

Go (in VirtualDJ) to settings

Move to Audio Device sheet and select Advanced Config, click to right button.

Set like this: 

 Left Deck
 BCD2000 ASIO - Channel 1
 BCD2000 ASIO - Channel 2
 Righ Deck
 BCD2000 ASIO - Channel 1
 BCD2000 ASIO - Channel 2
 Headphone
 BCD2000 ASIO - Channel 3
 BCD2000 ASIO - Channel 4
 Samples
 BCD2000 ASIO - Channel 1
 BCD2000 ASIO - Channel 2
 Prelisten
 BCD2000 ASIO - Channel 3
 BCD2000 ASIO - Channel 4

This is important to use headphone on BCD2000.

Know Problem or Features

The controller turnable of effect is not assigned.

The assigned control is sometimes differents from originals (but i prefer this one)

The Headphone Crossfader is set as a Mix (but i prefer this one)

Search func as Skip (someone can help me)

Button + and - has another function.
